# wikipedia_Covid19Pageviews

Dashboard with Daily Pageviews and Daily Pageviews line graph, data is calculated based on a curated list of the most relevant Covid 19 articles on Wikipedia in multiple languages/projects.

The strongly-related pages list is compiled by Diego (WMF) using all Wikidata Articles that link to a main COVID-19 pages COVID-19 (Q84263196) and the 2019â€“20 COVID-19 pandemic (Q81068910), then collecting relationships between the seeds and all the resulting items. The set we are using excludes pages specifically about people and other less relevant pages.
